# VibrationLab - Interactive Linear and Nonlinear Vibration Simulator

**VibrationLab** is a browser-based interactive simulator for exploring vibration, damping, resonance, nonlinear stiffness, and dynamic response of a single-degree-of-freedom mass-spring-damper system.

The project was created as an educational and engineering visualization tool. It helps users understand how system parameters affect vibration response, resonance amplification, energy exchange, frequency content, and nonlinear behavior.

---

## Governing Equation

The simulator is based on the forced nonlinear vibration equation:

m x'' + c x' + kx + alpha x^3 = F0 sin(omega t)

where:

| Symbol | Description |
|---|---|
| `m` | Mass |
| `c` | Damping coefficient |
| `k` | Linear stiffness |
| `alpha` | Cubic nonlinear stiffness coefficient |
| `F0` | Excitation force amplitude |
| `omega` | Excitation frequency |
| `x` | Displacement |
| `x'` | Velocity |
| `x''` | Acceleration |

When `alpha = 0`, the system is linear.

When `alpha > 0`, the system has hardening nonlinear stiffness.

When `alpha < 0`, the system has softening nonlinear stiffness.

## Derived Quantities

The app automatically calculates important vibration parameters.

Natural frequency:

omega_n = sqrt(k / m)

Damping ratio:

zeta = c / (2 sqrt(k m))

Critical damping coefficient:

c_crit = 2 sqrt(k m)

Damped natural frequency for underdamped systems:

omega_d = omega_n sqrt(1 - zeta^2)

Quality factor:

Q = 1 / (2 zeta)

Linear steady-state amplitude:

X(omega) = F0 / sqrt((k - m omega^2)^2 + (c omega)^2)

---

## Numerical Method

The second-order equation is converted into a first-order state-space system:

x_dot = v

v_dot = (F0 sin(omega t) - c v - k x - alpha x^3) / m

The system is solved numerically using a fixed-step fourth-order Runge-Kutta method.

---

## Energy Expression

The total mechanical energy is calculated as:

E = 0.5 m v^2 + 0.5 k x^2 + 0.25 alpha x^4

This includes:

- Kinetic energy
- Linear potential energy
- Nonlinear potential energy

---

## Force Balance

The equation of motion can be interpreted as a force balance:

m x'' = F0 sin(omega t) - c x' - kx - alpha x^3

The simulator visualizes:

- External excitation force
- Damping force
- Linear and nonlinear restoring force
- Inertial force

---

## Linear Frequency Response

For the linear case, where `alpha = 0`, the steady-state displacement amplitude is:

X(omega) = F0 / sqrt((k - m omega^2)^2 + (c omega)^2)

This curve is used to visualize resonance and the influence of damping.

For nonlinear systems, this curve is shown only as a linear reference.

---

## Transmissibility

The simulator includes a linear force transmissibility reference.

This is useful for understanding how vibration transmission changes with frequency ratio:

r = omega / omega_n

The transmissibility curve is exact for the linear system and should be interpreted as a reference when nonlinear stiffness is active.

---

## Poincare Section

The Poincare section is generated by sampling the displacement and velocity once per forcing period.

This helps visualize whether the forced response is:

- Periodic
- Multi-periodic
- More complex or irregular

For a simple periodic response, the Poincare points tend to cluster near a small number of locations.

---

## Bifurcation Diagram

The app includes a bifurcation diagram for nonlinear forced vibration.

The bifurcation diagram is generated by:

1. Sweeping the excitation frequency.
2. Simulating the nonlinear oscillator at each frequency.
3. Removing transient response.
4. Sampling displacement once per forcing period.
5. Plotting sampled displacement versus excitation frequency.

This is useful for visualizing nonlinear Duffing-type behavior, including complex periodic responses and possible branch splitting.

The bifurcation diagram is intended as an educational visualization. It is a browser-friendly numerical approximation, not a full continuation-analysis tool.

## Important Notes and Limitations

The time-domain simulation includes nonlinear stiffness when `alpha` is not zero.

The analytical resonance curve and transmissibility curve are linear references and are exact only when `alpha = 0`.

For nonlinear systems, response may depend strongly on:

- Excitation amplitude
- Initial conditions
- Damping
- Nonlinear stiffness
- Simulation time
- Frequency sweep settings

The bifurcation diagram is approximate and depends on the selected numerical sweep settings.

This project is intended as an educational and visualization tool. It is not certified engineering design software.
